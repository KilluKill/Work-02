# Основы HTML
- html 5 - текущий
- около 100 тэгов (принцип 20-80)
- признак тэга <>
- тэги бывают парные и непарные
- тэги управляются или дополняются атрибутами
- заголовки - h1 - h6
- парагафы - p
- изображение - img
- списки - ul - li, ol - li
- симантические тэги
- два самых популярных тэга - div и span
